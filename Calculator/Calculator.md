# 🧮 Calculator - Java Console App

Welcome to **Java Console Calculator**, a beginner-friendly console-based calculator built using Java. This application allows users to perform basic arithmetic operations interactively using a text-based interface.

---

## 📋 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division (with zero-division handling)
* 🟰 Square of a number
* 🧮 Option to perform **all operations** at once
* 🔁 Runs continuously until user exits
* 🧾 Cleanly structured `switch-case` logic for handling user commands
* 🧠 Educational and beginner-friendly

---

## 🛠 Technologies Used

* **Java SE (Standard Edition)**
* `Scanner` class for user input
* Console-based menu system
* Static methods for each operation

## 🚧 Known Limitations

* ❌ No input validation for non-numeric inputs
* ❌ Square function still prompts for a second number (though it ignores it)
* ❌ Does not handle complex mathematical operations (roots, powers, etc.)
* ❌ No GUI or real-time display
